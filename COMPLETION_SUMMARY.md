## 🚀 快速开始

### 1. 启动后端
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows
pip install -r requirements.txt
python -m uvicorn main:app --reload --host 0.0.0.0 --port 8001
```

### 2. 启动前端
```bash
cd frontend
npm install
npm run dev
```

### 3. 访问应用
- 前端：http://localhost:5173
- 后端：http://localhost:8001
- API 文档：http://localhost:8001/docs

## 📝 使用流程

### 记录想法
1. 打开首页
2. 点击"+"按钮
3. 输入想法内容
4. 选择分类（可选）
5. 保存

### 生成回顾
1. 打开回顾页面
2. 选择时间范围
3. 选择分类（可多选）
4. 点击"✦ AI 生成回顾"
5. 查看三种模式的洞察
6. 点击"保存回顾"

### 查看历史
1. 在回顾页面向下滚动
2. 查看"📚 历史回顾"列表
3. 点击查看详情

### 分享回顾
1. 生成回顾后
2. 点击"保存图片"
3. 图片自动下载

## 💡 使用建议

### 最佳实践
1. **定期记录**：养成记录想法的习惯
2. **及时分类**：创建想法时选择合适的分类
3. **定期回顾**：每周或每月生成一次 AI 回顾
4. **保存重要回顾**：将有价值的回顾保存下来
5. **定期备份**：备份数据库文件

### 使用技巧
1. 使用标签组织相关想法
2. 置顶重要的想法
3. 尝试不同的回顾模式
4. 分享精彩的回顾卡片
5. 定期清理不需要的想法
