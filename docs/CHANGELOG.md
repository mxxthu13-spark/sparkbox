# SparkBox v1.0 更新日志

## [1.0.0] - 2024-03-06

### 🎉 首次发布

SparkBox v1.0 正式发布！这是一个优雅的思想记录与 AI 回顾工具。

### ✨ 新增功能

#### 核心功能
- **想法管理**
  - 创建、编辑、删除想法
  - 按分类筛选
  - 关键词搜索
  - 标签管理
  - 置顶功能
  - 时间显示优化（显示"刚刚"、"5分钟前"等）

- **AI 回顾系统**
  - 摘要模式：提炼核心思考方向（150-200字）
  - 洞察模式：挖掘思想结构和价值观（150-200字）
  - 灵魂模式：深层内在对话（150-200字）
  - 长期主题识别
  - 自动分段排版

- **历史回顾**
  - 保存 AI 生成的回顾
  - 查看历史回顾列表
  - 显示时间范围、记录数、分类信息
  - 查看完整 AI 内容和原始想法
  - 删除回顾功能

- **分类管理**
  - 创建自定义分类
  - 图标和颜色自定义
  - 多分类筛选
  - 分类统计

- **分享功能**
  - 生成精美的回顾卡片
  - 保存为图片（自动隐藏按钮）
  - 统一的品牌设计

#### 用户界面
- **首页**
  - 层叠方块 Logo
  - 想法列表展示
  - 快速搜索
  - 统计信息

- **回顾页面**
  - 时间范围选择
  - 多分类筛选
  - 实时记录数量显示
  - 三种模式切换
  - 历史回顾列表（含分类标签）

- **回顾详情页面**
  - 简洁的布局设计
  - AI 内容展示
  - 原始想法列表
  - 底部分类信息
  - 删除功能

- **分享卡片**
  - 精美的卡片设计
  - 层叠方块 Logo
  - 统计信息展示
  - 一键保存图片

### 🎨 设计优化

- **Logo 设计**
  - 采用层叠方块图标（代表思想的积累和沉淀）
  - 统一的品牌色：#6366f1

- **排版优化**
  - AI 内容自动分段（2-3段）
  - 清晰的视觉层次
  - 舒适的阅读体验

- **交互优化**
  - 移除不必要的模式标签
  - 简化操作流程
  - 友好的提示信息

### 🔧 技术实现

#### 后端
- FastAPI 框架
- SQLite 数据库
- SQLAlchemy ORM
- JWT 认证
- DeepSeek/Qwen AI 集成
- 异步数据库操作

#### 前端
- Vue 3 + Vite
- Pinia 状态管理
- Vue Router 路由
- html2canvas 图片生成
- 响应式设计
